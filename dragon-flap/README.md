# Dragon Flap

Personal game development project built using HTML, CSS and JavaScript.

## Objective

Create a simple game while learning programming, debugging and project management.

## Technologies

- HTML
- CSS
- JavaScript

## Skills Demonstrated

- Problem solving
- Testing and debugging
- Iterative development
- Technical research
- AI-assisted development

## Lessons Learned

During development I encountered multiple issues related to Android exports, Java configuration and file path compatibility. Solving these issues helped me improve my troubleshooting and technical investigation skills.
