# Test Case TC-001

## Feature

Character Jump

## Objective

Verify that the player character jumps correctly when the jump key is pressed.

## Preconditions

* Game is running.
* Character is standing on the ground.

## Test Steps

1. Start the game.
2. Press the Space Bar.
3. Observe character behavior.

## Expected Result

The character performs a jump animation and moves upward.

## Actual Result

The character jumps correctly.

## Status

PASS
