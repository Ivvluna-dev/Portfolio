# Test Case TC-002

## Feature

Player Collision

## Objective

Verify that the player cannot pass through solid obstacles.

## Preconditions

* Game is running.
* Obstacles are present.

## Test Steps

1. Move character toward obstacle.
2. Continue holding movement key.

## Expected Result

The character stops at the obstacle.

## Actual Result

The character does not pass through the obstacle.

## Status

PASS
